
Wolfish HTML
======

Full README.md file soon. Well, probably  ¯\_(ツ)_/¯

First run:
---

Go to project folder and run in terminal window: 
```
npm install
```

Usage: 
---

__For development:__
```
gulp watch
```

__For deploy to production:__
```
gulp deploy
```
